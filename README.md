## Setup
1. Install dependencies:
   ```bash
   npm install

2. Configure database connection in .env:
DATABASE_URL="postgresql://user:password@localhost:5432/concentrator"

3. Run migrations:
npx prisma migrate dev

4. Start the app:
npm run start:dev
